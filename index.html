Keyboard Controls

  • W: Increase height
  • S: Decrease height
  • D: Increase width
  • A: Decrease width
  • Q: Rotate counterclockwise
  • E: Rotate clockwise
  • Shift+Q: Rotate 45 degrees counterclockwise
  • Shift+E: Rotate 45 degrees clockwise
  • R: Align with multiple of 45 degrees
  • L: Toggle flashlight or change panel color
  • K: Change panel mode (add, subtract, equal)
  • Shift+L: Change panel color in opposite direction
  • Shift+K: Change panel mode in opposite direction
  • 1: Add mode
  • 2: Edit mode
  • 3: Delete mode
